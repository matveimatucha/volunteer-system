# Google Sheets шаблон для заявок

Ниже готовый минимальный шаблон, чтобы работать с CSV из админки:

- лист `raw` — вставляете CSV как есть;
- лист `new_only` — автоматически показывает только новые строки;
- лист `analytics` — считает базовые метрики.

## 1) Лист `raw`

1. Создайте лист `raw`.
2. Вставьте CSV из админки (кнопка `Скачать все CSV` или `Скачать новые CSV`) в ячейку `A1`.
3. В первой строке должны быть колонки:
   - `registrationId`
   - `createdAt`
   - `createdAtMs`
   - `timestamp`
   - `eventId`
   - `eventTitle`
   - дальше поля анкеты

## 2) Лист `seen_ids`

1. Создайте лист `seen_ids`.
2. В `A1` напишите `registrationId`.
3. Сюда можно копировать id уже обработанных заявок (если хотите свой отдельный контроль помимо сайта).

## 3) Лист `new_only`

Создайте лист `new_only`, в `A1` вставьте формулу:

```gs
=LET(
  data, raw!A2:ZZ,
  ids, INDEX(data,,1),
  seen, FILTER(seen_ids!A2:A, seen_ids!A2:A<>""),
  FILTER(data, (ids<>"") * ISNA(MATCH(ids, seen, 0)))
)
```

Что делает:
- берет все строки из `raw`;
- исключает те, у которых `registrationId` уже есть в `seen_ids`;
- оставляет только новые.

## 4) Лист `analytics`

### 4.1 Всего заявок

В `A1`: `Всего заявок`  
В `B1`:

```gs
=COUNTA(raw!A2:A)
```

### 4.2 Уникальных мероприятий

В `A2`: `Уникальных мероприятий`  
В `B2`:

```gs
=COUNTA(UNIQUE(FILTER(raw!E2:E, raw!E2:E<>"")))
```

### 4.3 Новых заявок (по листу new_only)

В `A3`: `Новых заявок`  
В `B3`:

```gs
=COUNTA(new_only!A:A)
```

### 4.4 Таблица "Заявок по мероприятиям"

В `A5`:

```gs
=QUERY(
  raw!A1:F,
  "select F, count(A) where A is not null group by F label count(A) 'Заявок'",
  1
)
```

## 5) Рекомендуемый рабочий цикл

1. В админке нажмите `Скачать новые CSV`.
2. Добавьте строки в конец листа `raw` (не перезатирая старые).
3. После обработки заявок скопируйте `registrationId` в `seen_ids`.
4. Лист `new_only` и блок `analytics` обновятся автоматически.

---

Если хотите, следующим шагом можно добавить Google Apps Script, который:
- по кнопке импортирует CSV из файла на Google Drive;
- автоматически дописывает в `raw`;
- обновляет `seen_ids` после отметки обработанных строк.
